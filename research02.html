Modified Linear Selection Algorithm [pdf]

Yanchao Sun, Yu Chen

Abstract The linear selection algorithm is one of the classic algorithm in divide-and-conquer algorithm. It can resolve the classic problem within linear time complexity which selected the k-th smallest element in the set of n elements. However, when the value of k is smaller, the strategy by selecting the median of the median as the pivot element in the classic algorithm is inefficiency. To tackle this problem, in this study, we improved the classic linear selection algorithm. A novel method to select the pivot element dynamical is proposed, which determines the best pivot element by comparing the median and the value of k in every iteration. Based on this method, the search space can be pruned dramatic. Extensive experiments illustrate that our new method outperforms the classic algorithm. Moreover, in some cases the efficiency can be improved several times.

My contribution
1. Proposing the idea.
2. Designing the algorithm.
3. Conducting the experiments of our method and the baseline method.
4. Writing most of the paper.
